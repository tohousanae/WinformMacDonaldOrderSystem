# 模擬麥當勞訂餐
## 功能

測試帳號密碼 **（請斟酌提供，建議只提供僅能觀看不能操作的帳號密碼）**

```bash
帳號： example@example.com
密碼： example
```
客人端功能：
- [x] 會員登入
- [x] 會員註冊
- [x] 修改會員資料
- [x] 瀏覽商品(可選圖片模式或文字模式)
- [x] 加入購物車、送出訂單(結帳)、輸出明細

商家端功能：
- [x] 新增商品
- [x] 修改商品
- [x] 刪除商品
...
